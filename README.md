# Marvel_TriviaMaze
A maze where each door has a Marvel trivia question and the player has to go through it to find the finish exit.

![image](https://github.com/atofus/Marvel_TriviaMaze/assets/138819170/94821a31-e473-4071-804d-2bb6ceacf8b7)

Above is a picture of what the main game interface looks like. The user's goal is to reach the bottom left corner of the screen. To go through a door and unlock that door, they have to answer a Marvel trivia question correctly, otherwise, the door will be forever locked. If the user is unable to reach the end, the game will end and the game will show the user's score and put it on the leaderboard.
